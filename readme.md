# Wykop User Informator

Pobiera dane użytkoników wykopu.pl bez uzycia API.

###Funkcjonalności
- pobieranie wszystich komentarzy z oceną i datą
- sortowanie komentarzy względem oceny
- zapis do pliku

###Użycie

```bash
python run.py {username}
```
