# Wykop User Informator

Pobiera dane użytkowników wykopu.pl bez uzycia API.

### Funkcjonalności
- pobieranie wszystkich komentarzy z oceną i datą
- sortowanie komentarzy względem oceny
- zapis do pliku

### Użycie

```bash
python run.py {username}
```
